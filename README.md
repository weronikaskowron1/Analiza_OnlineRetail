# Analiza bazy danych sprzedaży detalicznej online  

Wielowymiarowa analiza danych sprzedażowych brytyjskiego sklepu internetowego obejmująca redukcję wymiarowości, klasteryzację produktów, analizę zależności ceny od wolumenu sprzedaży oraz odkrywanie reguł asocjacyjnych w koszykach zakupowych klientów.  

## Technologie  
- Python  
- Pandas, NumPy  
- Scikit-learn  
- Matplotlib, Seaborn, Plotly  
- mlxtend (Apriori)  

## Zakres prac  
- Czyszczenie i przygotowanie danych sprzedażowych  
- Agregacja danych na poziomie produktów  
- Standaryzacja danych  
- Redukcja wymiarowości metodą PCA  
- Klasteryzacja produktów (KMeans) + ocena silhouette  
- Analiza zależności ceny od wolumenu sprzedaży (regresja liniowa i logarytmiczna)  
- Analiza koszyków zakupowych i odkrywanie reguł asocjacyjnych (Apriori)  
- Wizualizacja wyników (wykresy PCA, klastry, regresja, reguły asocjacyjne)
